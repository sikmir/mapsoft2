Mapsoft2

A collection of tools and libraries for working with maps and geo-data.

About

We started writing programs for handling GPS-data and maps since first GPS-receivers appeared in our hiking group at 1997-1998. First it was tools for downloading tracks, putting them on a map, editing, and uploading back to GPS. When we started creating our own maps: hiking maps of Moscow region and some mountain regions:
  • Maps of Moscow region at nakarte.me
  • About Moscow region maps (RU)
  • Mountain maps at nakarte.me

All programs for working with geodata, raster and vector maps, SRTM-data, etc, have been collected to mapsoft project.

Mapsoft-2 is an attempt to rewrite these programs fixing some fundamental problems and improving the code.

Source code library

Mapsoft consists of many different pieces of code: simple and sophisticated, stable and experimental, good and bad. Some of this pieces are also used in other projects, not related to maps. I found very useful to make a "source code library" which consists of separate "modules" (something like gnulib). Each module has its own documentation, tests, and it can be easily transferred to another project.

Programs

  •   ms2conv -- Converter for geodata and raster maps .
  •   ms2view -- Viewer for geodata and raster maps.
  •   ms2nom -- Program for soviet nomenclature map calculations. Coordinates of a map sheet, adjacent sheets, sheets covering a given region, etc.
  •   ms2tiles -- Program for TMS/Google tile calculations. Coordinates of a tile, tiles covering a given region, etc.
  •   ms2xyz -- Program for writing tracks as text tables. Time, distance, velocity calculations, adjustable format.
  •   ms2ref -- Program for making map references. Only OziExplorer map output is supported now.
  •   ms2vmap -- Program for working with vector maps.

Here   - more or less stable programs,   - unstable programs, interface will be changed,   - unusable programs.

Supported file formats

  • Geodata: OziExplorer, GPX, KML, GarminUtils, GeoJSON.
  • Raster images: JPEG, PNG, TIFF, GIF.
  • FIG - vector graphics format
  • MP - "Polish" format for vector maps
  • VMAP - mapsoft1 format for vector maps
  • SHP+DBF - format for vector map data
  • MapDB -- Mapsoft2 native format for vector maps.
