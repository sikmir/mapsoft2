NAME

ms2vmap -- working with vector maps

SYNOPSIS

        ms2vmap (-h|--help|--pod)

        ms2vmap <action> (-h|--help)

        ms2vmap <action> [<action arguments and options>]

GENERAL OPTIONS:

-h, --help

Show help message.

--pod

Show help message as POD template.

ACTIONS:

        ms2vmap import_mp <arguments> -- import MP file to MapDB

        ms2vmap export_mp <arguments> -- export MP file from MapDB

        ms2vmap import_vmap <arguments> -- import mapsoft1 VMAP file to MapDB

        ms2vmap export_vmap <arguments> -- export mapsoft1 VMAP file from MapDB

        ms2vmap render <arguments> -- Render image from MapDB

IMPORT_MP -- IMPORT MP FILE TO MAPDB

EXPORT_MP -- EXPORT MP FILE FROM MAPDB

IMPORT_VMAP -- IMPORT MAPSOFT1 VMAP FILE TO MAPDB

EXPORT_VMAP -- EXPORT MAPSOFT1 VMAP FILE FROM MAPDB

RENDER -- RENDER IMAGE FROM MAPDB
