NAME

ms2vmap -- working with vector maps

SYNOPSIS

        ms2vmap <vmap> (-h|--help|--pod)

        ms2vmap <vmap> <action> (-h|--help|--pod)

        ms2vmap <vmap> <action> [<action arguments and options>]

GENERAL OPTIONS:

-h, --help

show help message

--pod

show this message as POD template

ACTIONS:

 * import_mp <name> <options> -- import MP file to the map
 * export_mp <name> <options> -- export the map to MP file
 * import_vmap <name> <options> -- import VMAP1 file to the map
 * export_vmap <name> <options> -- export the map to VMAP1 file
