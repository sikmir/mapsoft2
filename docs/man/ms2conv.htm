NAME

ms2conv -- mapsoft2 converter for geodata and raster maps

USAGE

        ms2conv <options> <input files> -o <output file>

GENERAL OPTIONS:

-h, --help -- show help message

--pod -- show help message as POD template

-v, --verbose -- be verbose

READING AND WRITING GEODATA

See GEODATA FORMATS section in mapsoft2(5) for supported formats and related options.

SETTING UP MAP REFERENCE FOR RASTER IMAGES

See SETTING UP A MAP REFERENCE section in mapsoft2(5) for related options.

DRAWING OPTIONS FOR RASTER IMAGES

See RENDERING GEODATA and RASTER IMAGE FORMATS sections in mapsoft2(5) for related options.
